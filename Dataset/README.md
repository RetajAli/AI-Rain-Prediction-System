
# Dataset

This folder contains the weather dataset used to train and evaluate the machine learning models.

## Contents

- train_weather.csv

## Features

The dataset includes weather-related attributes such as:

- Temperature
- Humidity
- Pressure
- Wind Speed
- Wind Direction
- Rainfall
- Location

## Target Variables

- RainToday
- RainTomorrow

The dataset was used for binary classification tasks to predict rainfall occurrence.
